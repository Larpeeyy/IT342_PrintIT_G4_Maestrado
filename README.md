# PrintIT

PrintIT is a web-based and mobile-ready printing management system for students and campus printing services. It helps students submit printing requests online and helps staff manage orders in a more organized way.

## Features

- User registration and login
- Google OAuth login
- Role-based access for Student, Staff, and Admin
- Secure password encryption using BCrypt
- Spring Boot backend API
- PostgreSQL database using Supabase

## Tech Stack

**Frontend**
- React
- React Router
- Axios

**Backend**
- Spring Boot
- Spring Security
- JPA / Hibernate

**Database**
- Supabase PostgreSQL
